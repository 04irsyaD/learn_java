# 🧮 Calculator App

Project sederhana untuk mengimplementasikan dasar-dasar Java (variabel, operator, percabangan, perulangan) dengan membuat kalkulator interaktif di terminal.

## Fitur
- Operasi dasar: Tambah (+), Kurang (-), Kali (*), Bagi (/)
- Operasi tambahan: Modulus (%), Pangkat (^)
- Penanganan error pembagian dengan nol.
- Operasi dapat dilakukan berulang kali tanpa harus me-restart program.

## Cara Menjalankan
1. Buka terminal/command prompt.
2. Navigasi ke direktori `projects/calculator-app`.
3. Compile file Java:
   ```bash
   javac Main.java Calculator.java
   ```
4. Jalankan program:
   ```bash
   java Main
   ```
