# ✈️ Travel Booking System

Sistem pemesanan tiket perjalanan sederhana menggunakan Java Collections dan OOP.

## Fitur
- Kelola Destinasi Wisata (Nama, Lokasi, Harga).
- Registrasi Customer.
- Booking tiket perjalanan.
- Pembatalan tiket perjalanan.
- Cetak struk/receipt pemesanan ke terminal.

## Cara Menjalankan
1. Compile file Java:
   ```bash
   javac Destination.java Customer.java Booking.java BookingManager.java Main.java
   ```
2. Jalankan program:
   ```bash
   java Main
   ```
