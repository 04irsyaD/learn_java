# 📚 Library Management System

Sistem manajemen perpustakaan sederhana menggunakan Object-Oriented Programming (OOP) di Java.

## Fitur
- Kelola data buku (Tambah, Lihat).
- Kelola data anggota (Tambah, Lihat).
- Peminjaman buku oleh anggota (mengecek status buku).
- Pengembalian buku.
- Menampilkan riwayat peminjaman.

## Cara Menjalankan
1. Compile file Java:
   ```bash
   javac Book.java Member.java Library.java Main.java
   ```
2. Jalankan program:
   ```bash
   java Main
   ```
