# 🎓 Student Management System

Project ini merupakan simulasi sistem manajemen data mahasiswa menggunakan Java Collections (ArrayList).

## Fitur
- Create (Tambah Mahasiswa)
- Read (Lihat Data Mahasiswa, Cari Mahasiswa)
- Update (Ubah Data Mahasiswa)
- Delete (Hapus Mahasiswa)
- Sorting mahasiswa berdasarkan nama atau IPK.

## Cara Menjalankan
1. Compile file Java:
   ```bash
   javac Student.java StudentManager.java Main.java
   ```
2. Jalankan program:
   ```bash
   java Main
   ```
