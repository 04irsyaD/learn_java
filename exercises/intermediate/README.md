# 🟡 Latihan: Level Intermediate

Kumpulan soal latihan untuk melatih logika perulangan, penggunaan array, dan pembuatan method.

## 1. Pola Segitiga Bintang
**Deskripsi:** Buat program untuk mencetak pola segitiga menggunakan bintang `*` sebanyak N baris.
**Hint:** Gunakan nested `for` loop.

## 2. Nilai Maksimum dan Minimum Array
**Deskripsi:** Cari nilai terbesar dan terkecil dari sebuah array berisi bilangan bulat.
**Hint:** Gunakan iterasi loop melalui elemen-elemen array.

## 3. Kalkulator Faktorial
**Deskripsi:** Buat method untuk menghitung nilai faktorial dari sebuah angka (misal: 5! = 5x4x3x2x1).
**Hint:** Bisa menggunakan loop atau rekursi.

## 4. Palindrome Checker
**Deskripsi:** Buat method untuk mengecek apakah sebuah kata adalah palindrome (dibaca dari depan dan belakang sama).
**Hint:** Gunakan manipulasi `String` dan array karakter.

## 5. Penjumlahan Matriks 2D
**Deskripsi:** Buat program untuk menjumlahkan dua buah matriks 2x2.
**Hint:** Gunakan array 2 dimensi dan nested loop.
