# 🟢 Latihan: Level Beginner

Kumpulan soal latihan untuk memantapkan pemahaman dasar Java, termasuk variabel, tipe data, operator, dan percabangan dasar.

## 1. Biodata Sederhana
**Deskripsi:** Buat program yang mencetak biodata diri menggunakan variabel dengan tipe data yang sesuai.
**Hint:** Gunakan `String` untuk nama, `int` untuk umur, dan `double` untuk IPK.

## 2. Kalkulator Diskon
**Deskripsi:** Buat program yang menghitung harga akhir setelah diskon. Jika total belanja > 100000, beri diskon 10%.
**Hint:** Gunakan `if-else` dan tipe data `double`.

## 3. Cek Bilangan Ganjil Genap
**Deskripsi:** Buat program yang menentukan apakah sebuah angka genap atau ganjil.
**Hint:** Gunakan operator modulus `%`.

## 4. Konversi Suhu
**Deskripsi:** Buat program untuk mengonversi suhu dari Celcius ke Fahrenheit, Kelvin, dan Reamur.
**Hint:** Gunakan rumus konversi matematika standar.

## 5. Cek Tahun Kabisat
**Deskripsi:** Buat program untuk mengecek apakah sebuah tahun merupakan tahun kabisat.
**Hint:** Tahun kabisat habis dibagi 4 tetapi tidak habis dibagi 100, atau habis dibagi 400.
