# 🔴 Latihan: Level Advanced

Kumpulan soal latihan untuk melatih Object-Oriented Programming (OOP), Collections, dan Exception Handling.

## 1. Sistem Akun Bank (OOP)
**Deskripsi:** Buat class `AkunBank` dengan enkapsulasi. Miliki method untuk setor tunai dan tarik tunai.
**Hint:** Gunakan access modifier `private` pada saldo dan public getter/setter/methods.

## 2. Hewan Peliharaan (Inheritance & Polymorphism)
**Deskripsi:** Buat class `Hewan` dan child class `Kucing` dan `Anjing` yang melakukan override pada method `bersuara()`.
**Hint:** Gunakan keyword `extends` dan `@Override`.

## 3. Data Mahasiswa (Collections)
**Deskripsi:** Buat program untuk menyimpan daftar mahasiswa beserta nilainya menggunakan `HashMap<String, Integer>`.
**Hint:** Pahami key-value pair.

## 4. Penghitung Kata (Collections)
**Deskripsi:** Buat program yang menerima kalimat dan menghitung frekuensi kemunculan tiap kata menggunakan `HashMap`.
**Hint:** Gunakan `.split()` pada String.

## 5. Penanganan Error Pembagian (Exception Handling)
**Deskripsi:** Buat program yang membagi dua angka yang diinput user, handle kasus jika pembagi adalah nol menggunakan exception.
**Hint:** Gunakan blok `try-catch` dan tangkap `ArithmeticException`.
