# 📦 Modul 08: OOP (Object-Oriented Programming)
Pemrograman Berorientasi Objek adalah paradigma pemrograman yang berfokus pada pembuatan objek yang berisi data (atribut) dan perilaku (method).

## Tujuan Pembelajaran
- Memahami konsep dasar OOP (Class dan Object)
- Memahami dan menerapkan 4 Pilar OOP: Encapsulation, Inheritance, Polymorphism, Abstraction
- Mengerti penggunaan Constructor dan Keyword `this`, `super`
- Mengenal Access Modifier (`public`, `private`, `protected`)

## Teori / Penjelasan
- **Class**: Blueprint atau cetakan untuk membuat objek.
- **Object**: Instansiasi nyata dari sebuah Class.
- **Encapsulation**: Menyembunyikan data internal objek dan menyediakan getter/setter.
- **Inheritance**: Pewarisan atribut dan method ke class anak menggunakan `extends`.
- **Polymorphism**: Kemampuan objek untuk mengambil berbagai bentuk (Overriding & Overloading).
- **Abstraction**: Menyembunyikan implementasi kompleks, menggunakan `abstract class` atau `interface`.

## Contoh Program
- `ClassObject.java`: Definisi class dasar dan instansiasi object.
- `Constructor.java`: Contoh penggunaan constructor.
- `Encapsulation.java`: Contoh private field dengan getter setter.
- `Inheritance.java`: Pewarisan antar class.
- `Polymorphism.java`: Overriding dan overloading.
- `Abstraction.java`: Abstract class dan interface.
